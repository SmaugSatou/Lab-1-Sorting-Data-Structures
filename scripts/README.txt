BENCHMARK VISUALIZATION SCRIPT
==============================

Requirements:
- Python 3.x
- pandas, matplotlib, numpy

Installation:

  Ubuntu/Debian:
    sudo apt install python3-pandas python3-matplotlib python3-numpy

  Arch Linux:
    sudo pacman -S python-pandas python-matplotlib python-numpy

  Using pip:
    pip install -r requirements.txt

Usage:

  From project root:
    python3 scripts/plot_results.py

  Or directly:
    ./scripts/plot_results.py

Output:
  Generates 6 plots in results/plots/ directory
